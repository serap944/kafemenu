# ☕ Kafe Menü & Sepet Sistemi

Bu proje, Vanilla JavaScript kullanılarak geliştirilmiş dinamik bir kafe menü ve sepet uygulamasıdır.

## 🚀 Özellikler

- 📂 Kategori bazlı ürün listeleme
- ➕ Ürünleri sepete ekleme
- 🔢 Ürün adet artırma / azaltma
- 🗑 Ürünü sepetten tamamen silme
- 💾 LocalStorage ile veri kalıcılığı
- 💰 Anlık toplam fiyat hesaplama
- 🔄 Dinamik render sistemi (DOM manipulation)
- 🧩 Component mantığı (Header yapısı)

## 🛠 Kullanılan Teknolojiler

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- LocalStorage API

## 🧠 Öğrenilen Konular

- DOM Manipülasyonu
- Event Delegation
- State Management (localStorage)
- Render mantığı
- Component yapısı (modüler JS)
